# Analisis-Exploratorio-y-Visualizacion-de-Datos-de-Ventas-con-Python-EDA-
Análisis Exploratorio de Datos (EDA) de ventas con Python. Usé visualización para detectar outliers, estacionalidad y la dominancia regional (SCANNING 50%). Proyecto enfocado en transformar datos limpios en insights de negocio y planeación comercial.
